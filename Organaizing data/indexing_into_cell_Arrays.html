Index into Cell Arrays

This code creates a cell array.
regions = { 'South America' 'Europe' ; 'Argentina' {'England' ; 'Italy' ; 'Netherlands'} }
regions = 2×2 cell

Task 1

Create a 2-by-1 cell array named col1 containing the first column of regions.
col1=regions(:,1)

Task 2

Extract the contents from the second row and second column of regions into a variable named ec. Note that this cell of regions contains another cell array. Hence, ec will be a 3-by-1 cell array.
ec=regions{2,2}
ec = 3×1 cell

'Netherlands'
